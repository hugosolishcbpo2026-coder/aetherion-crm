# AETHERION — AI Business Operating System
**Enterprise CRM for Notaría Pública 322 & TropicCo Property Management**

## 🚀 Quick Launch (GitHub Pages)

### Step 1 — Download files
Download `index.html` from this conversation.

### Step 2 — Create GitHub repository
1. Go to **github.com** → click **New repository**
2. Name it: `aetherion-crm` (or any name)
3. Set to **Public**
4. Click **Create repository**

### Step 3 — Upload file
1. Click **uploading an existing file**
2. Drag & drop `index.html`
3. Click **Commit changes**

### Step 4 — Enable GitHub Pages
1. Go to repo **Settings** → **Pages** (left sidebar)
2. Source: **Deploy from a branch**
3. Branch: **main** → folder: **/ (root)**
4. Click **Save**

### Step 5 — Access your CRM
Your URL will be:
```
https://YOUR-USERNAME.github.io/aetherion-crm/
```
Ready in ~2 minutes after saving.
